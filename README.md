# Voice of Iran

A documentation and awareness project focused on ongoing events in Iran — preserving memory, presenting verified narratives, and amplifying voices that have been silenced.

---

## Overview

This project presents:
- A structured timeline of events
- Context on protests, restrictions, and state actions
- Documented individual stories
- A growing archive of victims and affected individuals

The goal is not commentary, but **visibility and preservation**.

---

## Purpose

- Document events in a clear and accessible format  
- Preserve stories that risk being forgotten  
- Provide a resource that can be shared globally  
- Support awareness through structured information  

---

## Features

- Responsive web design (mobile + desktop)
- Timeline-based navigation
- Multilingual support (EN / DE)
- Dynamic rendering via JSON (stories & victims)
- Modular section structure for scalability

---

## Contributing

Contributions are welcome, especially for:

- Content accuracy improvements
- Translations (additional languages)
- UI/UX improvements
- Performance optimization
- Accessibility enhancements

### Guidelines

- Keep changes **factually grounded**
- Maintain **tone consistency** (neutral, documentation-focused)
- Do not introduce unverifiable claims
- Keep structure clean and minimal

Submit contributions via Pull Requests.

---

## Data Integrity

This project aims to present information responsibly.

- Numbers and claims should be treated as **reported estimates**
- Sources should be verifiable where possible
- Content may evolve as new information becomes available

---

## Anonymity

This project is maintained anonymously for security reasons.
